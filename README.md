# Cannabis Retail Sales Data Analysis

## Project Overview
This project analyzes cannabis retail sales data using Python to identify trends in sales, product demand, and pricing over time. The analysis uses real-world weekly retail sales data and visualizes patterns using data visualization techniques.

## Dataset
The dataset contains weekly cannabis retail sales information including:

- Week Ending
- Adult-Use Retail Sales
- Medical Marijuana Retail Sales
- Total Adult-Use and Medical Sales
- Total Products Sold
- Average Product Price

## Technologies Used

- Python
- Pandas
- Matplotlib
- VS Code

## Analysis Performed

The project performs the following analysis:

1. **Total Sales Trend**
   - Visualizes overall cannabis sales over time.

2. **Adult-Use vs Medical Sales**
   - Compares revenue trends between adult-use and medical cannabis markets.

3. **Product Sales Volume**
   - Analyzes how many products are sold weekly.

4. **Price Trend Analysis**
   - Examines changes in average cannabis product prices.

5. **Basic Statistical Summary**
   - Provides descriptive statistics for the dataset.

## Example Insights

- Adult-use cannabis sales account for the majority of retail revenue.
- Sales trends fluctuate over time, potentially due to seasonal demand.
- Product sales volume strongly correlates with total sales revenue.
- Average product prices change over time depending on market conditions.
